# 三国霸业 - 重置版

## 项目概述

**三国霸业-重置版** (Sanguo Baye Remastered) 是一款基于原版C++《三国霸业》游戏逻辑，使用现代Web技术重构的回合制策略战旗游戏。

## 技术栈

- **前端**: HTML5 + JavaScript (ES6+)
- **渲染**: Canvas API
- **状态管理**: 自定义状态管理器
- **架构**: MVC + 组件化 + 事件驱动

## 项目结构

```
sanguobaye-remastered/
├── docs/                    # 设计文档
├── data/                    # 游戏数据 (38城/200将/4时期)
├── src/                     # 源代码
│   ├── core/               # 核心系统 (引擎/事件/状态)
│   ├── data/               # 数据层 (仓库)
│   ├── models/             # 数据模型
│   ├── systems/            # 游戏系统
│   │   ├── internal/       # 内政系统
│   │   ├── diplomacy/      # 外交系统
│   │   ├── military/       # 军事系统
│   │   └── battle/         # 战斗系统
│   ├── utils/              # 工具类
│   ├── ui/                 # UI层
│   │   ├── components/     # 通用组件
│   │   ├── screens/        # 游戏界面
│   │   └── renderers/      # 渲染器
│   └── assets/             # 静态资源
├── tests/                  # 测试文件
└── tools/                  # 开发工具
```

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 运行测试
```bash
npm test
```

## 游戏数据

- **38座城池**: 包含农业/商业上限、人口、连接关系
- **200名武将**: 包含武力/智力/性格/兵种等属性
- **4个历史时期**: 董卓弄权、曹操崛起、赤壁之战、三国鼎立
- **16种技能**: 攻击/控制/辅助/特殊
- **24种道具**: 武器/防具/坐骑/兵书
